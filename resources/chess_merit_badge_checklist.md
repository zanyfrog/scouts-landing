# Chess Merit Badge Readiness Checklist

_Last updated: 2026-04-26_

This checklist is for Scouts to track when they believe they are ready to discuss, demonstrate, or complete each Chess merit badge item with a counselor. Checking an item here does not replace counselor approval.

Source: [Scouting America Chess Merit Badge requirements](https://www.scouting.org/merit-badges/chess/)

## 1. Chess History and Famous Players

**Completion rule:** Do all items in this section.

- [ ] I am ready to discuss the history of chess with my counselor.
- [ ] I have researched a famous chess player.
- [ ] I can explain what made that player famous.
- [ ] I am ready to discuss that player with my counselor.

## 2. Planning, Benefits, and Etiquette

**Completion rule:** Do all items in this section.

- [ ] I can explain why chess is a game of planning and strategy.
- [ ] I can discuss how chess develops critical thinking.
- [ ] I can discuss how chess develops concentration.
- [ ] I can discuss how chess develops decision-making.
- [ ] I can explain how those skills can help in other areas of life.
- [ ] I can discuss good sportsmanship in chess.
- [ ] I can discuss chess etiquette.

## 3. Teaching Chess Basics with Scouting EDGE

**Completion rule:** Do all items in this section.

- [ ] I know the name of each chess piece.
- [ ] I can correctly set up a chessboard.
- [ ] I can explain how each piece moves.
- [ ] I can explain how each piece captures.
- [ ] I can explain the four castling rules.
- [ ] I can explain en passant captures.
- [ ] I can explain pawn promotion.
- [ ] I can explain check.
- [ ] I can explain ways to get out of check.
- [ ] I can explain checkmate.
- [ ] I can explain the five ways a chess game can end in a draw.
- [ ] I am ready to use Scouting EDGE to teach someone who does not know how to play chess.

## 4. Notation, Game Phases, Openings, and Mates

**Completion rule:** Do all items in this section.

- [ ] I can keep score using algebraic chess notation.
- [ ] I can explain the opening.
- [ ] I can explain the middle game.
- [ ] I can explain the endgame.
- [ ] I can explain four opening principles.
- [ ] I can demonstrate the first five moves of the Ruy Lopez.
- [ ] I can demonstrate the first five moves of the French Defense.
- [ ] I can demonstrate the first five moves of the Queen's Gambit Declined.
- [ ] I can demonstrate the first five moves of the Sicilian Defense.
- [ ] I can demonstrate Scholar's Mate on a chessboard.
- [ ] I can demonstrate Fool's Mate on a chessboard.
- [ ] I can demonstrate Legal Mate on a chessboard.
- [ ] I can demonstrate the Fried Liver Attack on a chessboard.
- [ ] I can demonstrate Noah's Ark Trap on a chessboard.

## 5. Strategy, Tactics, Checkmates, Endgames, and Problems

**Completion rule:** Do all main items in this section. For strategy, choose at least four. For tactics, choose at least five.

- [ ] I can explain four chess strategy elements from the requirement list.
- [ ] I can explain five chess tactics from the requirement list.
- [ ] I can force checkmate with White king on e1, White rooks on a1 and h1, and Black king on e5.
- [ ] I can force checkmate with White king on e1, White queen on d1, and Black king on e5.
- [ ] I can force checkmate with White king on e1, White rook on a1, and Black king on e5.
- [ ] With White king on d4, White pawn on e3, and Black king on e6, I can demonstrate how White to move can force promotion.
- [ ] With White king on d4, White pawn on e3, and Black king on e6, I can demonstrate how Black to move can force a draw.
- [ ] I can set up and solve five direct-mate problems provided by my counselor.

### Strategy Elements to Prepare

**Completion rule:** Choose at least four.

- [ ] Exploiting weaknesses
- [ ] Force
- [ ] King safety
- [ ] Pawn structure
- [ ] Space
- [ ] Tempo
- [ ] Clock management

### Tactics to Prepare

**Completion rule:** Choose at least five.

- [ ] Clearance sacrifice
- [ ] Decoy
- [ ] Discovered attack
- [ ] Double check
- [ ] Double attack
- [ ] Fork
- [ ] Interposing
- [ ] Overloading
- [ ] Overprotecting
- [ ] Pin
- [ ] Skewer
- [ ] Remove the defender
- [ ] Zwischenzug
- [ ] Zugzwang

## 6. Chess Tournaments

**Completion rule:** Do all items in this section.

- [ ] I can explain how chess tournaments are run.
- [ ] I can explain the Swiss system tournament format.
- [ ] I can explain the round robin tournament format.
- [ ] I can explain pairings for each round.
- [ ] I can explain time controls.
- [ ] I can explain the touch-move rule.
- [ ] I can explain tournament scoring.
- [ ] I can explain chess ratings.

## 7. Game Play Option

**Completion rule:** Choose one option: A, B, or C.

### Option A: Play Three Games

**Completion rule:** If choosing Option A, do all items in this option.

- [ ] I have played at least three games of chess with other Scouts and/or my counselor.
- [ ] I have score sheets for those games.
- [ ] I can replay the games from my score sheets.
- [ ] I am ready to discuss how I might have played each game differently.

### Option B: Play in a Scholastic Tournament

**Completion rule:** If choosing Option B, do all items in this option.

- [ ] I have played in a scholastic youth chess tournament.
- [ ] I have score sheets from the tournament.
- [ ] I can replay my tournament games with my counselor.
- [ ] I am ready to discuss how I might have played each tournament game differently.

### Option C: Organize and Run a Tournament

**Completion rule:** If choosing Option C, do all items in this option.

- [ ] I have organized a chess tournament with at least four other players.
- [ ] I have run the chess tournament.
- [ ] I made sure each competitor played at least two games.
